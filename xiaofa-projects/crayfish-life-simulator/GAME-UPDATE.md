# 龙虾人生模拟器 - 游戏寿命和结局系统重构

## 核心改动总结

### 1. 游戏寿命延长

-   **事件触发频率调整**：从每 10 个事件增加 1 岁改为每 5 个事件增加 1 岁
-   **效果**：游戏时长翻倍，玩家有更多时间体验不同的人生阶段

### 2. 普通人生的结局逻辑（非修仙）

```
60岁以下：
  - 任意属性归零 → 直接结束（各种意外事件）
  - 30%概率随机结束（意外身亡）

60-105岁：
  - 任意属性归零 → 直接结束
  - 50%概率自然寿终正寝

105岁：
  - 强制结束或触发修仙
  - 有修仙天赋：80%概率进入修仙
  - 属性达标（智力≥70，运气≥60）：30%概率进入修仙
  - 其他情况：5%概率进入修仙
```

### 3. 修仙线的三个渡劫关卡

```
300岁 - 筑基期渡劫：
  - 失败条件：智力<80 或 运气<70，或20%随机失败
  - 成功：寿命延长到500岁

500岁 - 金丹期渡劫：
  - 失败条件：智力<90 或 运气<80，或25%随机失败
  - 成功：寿命延长到800岁

800岁 - 元婴期渡劫：
  - 失败条件：智力<95 或 运气<90，或30%随机失败
  - 成功：寿命延长到980岁

980岁 - 九九天劫（最终渡劫）：
  - 失败条件：智力<100 或 运气<95，或35%随机失败
  - 成功：飞升成仙
```

### 4. 新增事件（共 12 个）

**中年阶段（40-60 岁）**：

-   `m1`: 中年的迷茫
-   `m2`: 身体的警告
-   `m3`: 职业的巅峰或衰落
-   `m4`: 与父母的最后时光

**老年阶段（65-105 岁）**：

-   `o1`: 老年的智慧
-   `o2`: 生死的思考
-   `o3`: 最后的冒险

**修仙中期事件**：

-   `c310`: 修仙中期的突破（200-299 岁）
-   `c510`: 金丹的异变（400-499 岁）
-   `c810`: 元婴的觉醒（600-799 岁）
-   `c820`: 仙人的气息显现（750-950 岁）

**980 岁终极事件**：

-   `c982`: 虚空中的对话
-   `c983`: 龙虾本体的觉醒
-   `c984`: 飞升后的世界

### 5. 新增结局（共 10 个）

**普通人生结局**：

-   `accident_death`: 意外身亡（任意年龄）
-   `natural_death`: 寿终正寝（60 岁以上）

**修仙失败结局**：

-   `cultivator_300_fail`: 筑基失败
-   `cultivator_500_fail`: 金丹破碎
-   `cultivator_800_fail`: 元婴陨落
-   `cultivator_980_fail`: 九九天劫失败

**修仙成功结局**（980 岁以上）\*\*：

-   `cultivator_immortal_basic`: 初入仙班（基础飞升）
-   `cultivator_immortal_supreme`: 虾中至尊（融合龙虾力量，需要 vitality≥100）
-   `cultivator_immortal_free`: 自由的仙人（拒绝天地束缚，需要 intelligence≥100, luck≥95）
-   `cultivator_immortal_guardian`: 人间守护者（回到人间，需要 charm≥80, vitality≥70）
-   `cultivator_immortal_explorer`: 仙界探险家（探索仙界，需要 intelligence≥90, luck≥85）

### 6. 代码改动文件

1. **game.js**：

    - 修改事件触发频率（10→5）
    - 完善结局判断逻辑
    - 添加修仙阶段的年龄判断

2. **events.js**：

    - 添加 12 个新事件
    - 覆盖中年、老年、修仙各阶段

3. **endings.js**：

    - 替换旧结局为新结局
    - 添加修仙相关的详细结局描述

4. **result.js**：
    - 完善结局匹配逻辑
    - 根据修仙属性判断具体的修仙结局类型
    - 优化分享文案

## 游戏体验改进

✅ **寿命更长**：普通人生最长 105 岁，修仙线可达 980 岁
✅ **选择更多**：中年和老年阶段有更多事件选择
✅ **结局更丰富**：从原来的 13 个增加到 23 个结局
✅ **修仙线更完整**：从 105 岁开始，经历 300/500/800/980 四个渡劫关卡
✅ **属性影响更大**：修仙成功后的具体结局由属性决定
✅ **故事更连贯**：每个阶段都有对应的事件和结局

## 建议的后续优化

2. 可以添加天赋系统，让玩家在游戏开始时选择特殊天赋
3. 可以添加成就系统，记录玩家达成的各种结局
4. 可以添加排行榜，展示最高寿命等数据
