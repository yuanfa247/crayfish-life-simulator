# 《养一只小龙虾》微信小程序开发文档

文字人生模拟器 - 小龙虾养成内核

## 项目简介

这是一个微信小程序版文字人生模拟器，核心玩法：
- 开局随机生成小龙虾属性（活力、智力、财富、运气、颜值）
- 年龄增长触发随机事件，点击选择影响属性
- 最终根据属性和年龄触发不同结局，几十种结局等你体验
- 接入微信流量主，Banner + 激励视频广告变现

## 项目结构

```

├── miniprogram/

│   │   ├── game/           # 游戏主界面

│   │   └── settings/       # 设置页
│   ├── utils/              # 工具函数
│   └── images/             # 图片资源
├── project.config.json     # 项目配置
├── dev-plan.md             # 15天开发计划

```

## 命名规范

- 页面：小写+横线，如 `game/game`
- 变量：小驼峰，如 `currentAge`
- 函数：小驼峰，如 `triggerEvent`
- 常量：全大写下划线，如 `MAX_AGE`
