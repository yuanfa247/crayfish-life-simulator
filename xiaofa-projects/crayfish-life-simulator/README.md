# 《养一只小龙虾》微信小程序开发文档

文字人生模拟器 - 小龙虾养成内核

## 项目简介

这是一个微信小程序版文字人生模拟器，核心玩法：
- 开局随机生成小龙虾属性（活力、智力、财富、运气、颜值）
- 年龄增长触发随机事件，点击选择影响属性
- 最终根据属性和年龄触发不同结局，几十种结局等你体验
- 接入微信流量主，Banner + 激励视频广告变现

## 命名规范

- 页面：小写+横线，如 `game/game`
- 变量：小驼峰，如 `currentAge`
- 函数：小驼峰，如 `triggerEvent`
- 常量：全大写下划线，如 `MAX_AGE`

## 数据结构说明

### 属性系统
- `vitality` 活力：影响生存能力，活力归零游戏结束
- `intelligence` 智力：影响事件选择收益，触发特殊事件
- `wealth` 财富：影响经济相关事件结局
- `luck` 运气：影响事件随机概率，触发奇遇
- `charm` 魅力：影响社交相关事件

### 事件结构
```javascript
{
  title: "事件标题",
  description: "事件描述",
  ageMin: 0,      // 最小年龄
  ageMax: 5,      // 最大年龄
  requires: {      // 属性要求（可选）
    vitality: 20
  },
  tags: ["talent-name"], // 天赋标签（可选，对应天赋才能触发）
  options: [
    {
      text: "选项描述",
      effects: { vitality: +2, intelligence: -1 } // 属性变化
    }
  ]
}
```

### 天赋结构
```javascript
{
  name: "天赋名称",
  description: "天赋描述",
  grade: "purple", // 天赋等级：blue/purple/orange/red
  effects: { vitality: +3 }, // 属性加成
  tags: ["tag-name"], // 事件标签，对应标签事件概率提升
}
```
