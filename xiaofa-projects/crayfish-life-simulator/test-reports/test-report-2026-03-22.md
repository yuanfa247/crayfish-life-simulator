### 3.1 属性自然衰减机制 (新增)

**代码位置**: `game.js` 第 229-253 行

```javascript
ageDecay: function (attributes, age) {
    if (age < 40) return attributes; // 40岁前不衰减
    if (age > 105 && this.data.isCultivator) return attributes; // 修仙者不衰减

    // 40-60岁：20%概率衰减，60-80岁：40%，80+：60%
    let decayChance = 0.2;
    if (age >= 60) decayChance = 0.4;
    if (age >= 80) decayChance = 0.6;
    // ...
}
```

| 测试点 | 预期结果 | 实际结果 | 状态 |
|--------|----------|----------|------|
| 40岁前不衰减 | 无衰减 | 无衰减 | ✅ |
| 修仙者不衰减 | 无衰减 | 无衰减 | ✅ |
| 40-60岁衰减概率 | 20% | 20% | ✅ |
| 60-80岁衰减概率 | 40% | 40% | ✅ |
| 80+岁衰减概率 | 60% | 60% | ✅ |
| 衰减优先级 | 活力>魅力>财富>智力>运气 | 正确 | ✅ |
| 属性归零检测 | 触发结局 | 正确 | ✅ |

**结论**: 属性衰减机制设计合理，符合人类衰老规律。
