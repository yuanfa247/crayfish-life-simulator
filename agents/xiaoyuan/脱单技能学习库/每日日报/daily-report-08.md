## 今日内容

1. **见面之前准备**
   - 形象：干净整洁比穿名牌重要，头发剪好，胡子刮干净，衣服无异味
   - 地点：选人多的公共场所（商场咖啡店/餐厅），安全第一，避免偏僻地方
   - 时间：选下午或晚上，控制在1-2小时，不要第一次就耗一整天
   - 预算：提前查好大概消费，心里有数，主动买单，不要让女生付

2. **见面过程核心注意事项**
   - ❶ 准时：提前10分钟到，千万不要迟到
   - ❷ 主动：主动开门、拉椅子、点餐，适度主动不妈宝
   - ❸ 手机：调静音，不要一直玩手机，除非回消息快速处理
   - ❹ 买单：主动买单，不要问"我买可以吗"，直接去买
   - ❺ 肢体：第一次见面不要有肢体接触，保持安全距离
   - ❻ 尊重：对方说的话题即使不感兴趣也不要打断，认真听

3. **聊什么话题（安全高赞话题）**
   - **兴趣爱好类**：平时休息喜欢做什么？最近有没有看什么好看的剧？
   - **美食探店类**：你平时喜欢吃什么口味？这附近你有没有常去的店？
   - **生活见闻类**：最近有没有去哪里玩？最近发现什么好玩的地方？
   - **工作趣事类**：你平时工作主要做什么？你们上班有意思吗？（不要问工资）
   - **旅行见闻类**：你去过哪些城市？最喜欢哪个地方？

4. **绝对不要聊的雷区话题**
   - ❶ 不要问情感经历："谈过几次恋爱""为什么分手"
   - ❷ 不要查户口：年龄/工资/体重/房车，这些第一次不要问
   - ❸ 不要负能量抱怨：抱怨工作、抱怨生活、抱怨前女友
   - ❹ 不要讲低俗段子：容易显得油腻不尊重人
   - ❺ 不要说教辩论：对方观点和你不同，听着就行，不用非要赢
   - ❻ 不要上来就聊未来规划："我想结婚生孩子"，压力太大

5. **聊天节奏技巧**
   - 7分听3分说，多让对方说，你多提问多倾听
   - 学会追问：对方说喜欢旅游 → "你最近一次旅游是去哪里？" → "那地方什么最好吃？"
   - 适当分享自己，不要只问不说，对话是双向的
   - 冷场了不要慌，可以找当下场景话题："这家店装修你觉得怎么样？""这个咖啡味道还不错"

6. **见面结束后怎么做**
   - 主动送女生到地铁/打车，说今天玩得很开心，慢走
   - 回家后可以发个消息："我到家了，今天挺开心的，谢谢你"
   - 如果感觉不错，第二天可以继续聊，两三天内再次邀约
   - 如果感觉不合适，礼貌说清楚，不要冷暴力消失
