## 今日内容

1. **微信开场的核心原则**
   - 目的不是马上约出来，而是**让对方愿意回复你**
   - 无需求感开场：不说"我喜欢你""做我女朋友吧"，先交朋友
   - 不给对方压力：话题开放，方便对方接话
   - 展示自己一点，而不是只问对方

2. **错误开场雷区（90%不回）**
   - 查户口式："你好，你叫什么？哪里人？多大了？"
   - 舔狗式："你长得真漂亮，可以认识一下吗？"
   - 开启困难模式："在吗？有空吗？可以聊聊吗？"
   - 没营养式："早啊""今天天气真好"
   - 目的性太强："找对象吗？处对象吗？"

3. **3种好用的开场模板（亲测有效）**

### 模板一：基于朋友圈内容开场（最推荐）
- **原理**：从对方发的内容切入，说明你认真看了，不是群发
- **公式**：赞美具体细节 + 问一个相关问题
- **例子**：
  - "你去XX探店那家店真的好吃吗？看图片流口水了🤤"
  - "你家猫好可爱！是什么品种呀？"
  - "这张照片是在XX公园拍的吗？我上周也去了！"
- **优点**：精准命中，回复率最高

### 模板二：轻松有趣开场（适合刚加）
- **原理**：简单幽默，化解尴尬，不容易错
- **例子**：
  - "你好你好！冒昧通过XX加的，我是那个喜欢养猫的程序员，打扰啦"（自我介绍+背景）
  - "来了来了，终于加上了，先说你好🙋‍♂️"
  - "发现我们有XX共同好友，过来打个招呼"
- **优点**：自然不突兀，适用于大多数情况

### 模板三：求助式开场（低门槛）
- **原理**：人都有帮助欲，问题简单对方愿意回答
- **例子**：
  - "想问一下你XX地方哪家火锅比较好吃呀？打算周末去"
  - "你朋友圈这个口红是什么色号呀？帮我朋友问的"
  - "XX景点门票需要提前预约吗？打算去玩"
- **优点**：门槛极低，只要问题不复杂基本都回

4. **开场后的推进技巧**
   - 对方回复后，聊3-5轮就要找共同点
   - 不要查户口，你回答完也要说自己的情况
   - 聊出兴趣点就深化，没兴趣就快速换话题
   - 聊天节奏：对方发多长你发多长，不要连发几十条
   - 3天内约线下，不要聊半年还没见面

5. **不同场景适配**
   - 相亲/朋友介绍：直接用模板二，说明身份
   - 线上加的同城：用模板一（看朋友圈）
   - 活动认识：用共同经历开场"今天活动那个speaker你觉得咋样？"
