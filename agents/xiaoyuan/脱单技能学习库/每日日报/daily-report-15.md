## 今日内容

1. **小红书账号基础设置**
   - 头像：用本人清晰正面生活照，和微信头像保持一致，真实可信
   - 昵称：用本名或简单好记的昵称，不要搞奇怪符号
   - 简介：公式 = 年龄+城市+职业+爱好+交友目的，示例："95后，成都，产品经理，喜欢爬山喝咖啡，找个有意思的女生一起玩"
   - 背景图：放一张风景照或生活场景照，干净就行

2. **征友笔记发什么内容**
   - **开头**：直接说目的，真诚不绕弯，比如"95后男生成都找对象，认真交友"
   - **自我介绍**：说清楚基本情况（年龄/身高/城市/工作/学历），说一两个爱好，说一下性格，简单真实就好
   - **对方要求**：说核心要求（年龄范围/城市/大概要求），不要列十几条，显得挑剔
   - **结尾**：放联系方式提示（"感兴趣可以私信"，小红书不让直接放微信，会违规）

3. **征友笔记模板（直接可用）**

```
【基本情况】
95年，178，本科，互联网产品经理，定居成都，有房有车
平时喜欢爬山/喝咖啡/看展览，周末会出去探店，偶尔做饭
性格偏外向，比较爱笑，不抽烟不喝酒

【希望你】
92-00年，定居成都，身高155+，本科以上
性格开朗好相处，有正经工作
不接受胖的， sorry

【关于我】
真心找对象，不是玩玩，合适可以往结婚发展
感兴趣可以直接私信，看到会回
照片私，骗子勿扰
```

4. **照片怎么放**
   - 放3-6张，不要只放一张
   - 第一张放清晰正脸照
   - 其他放生活照（旅行/探店/运动/做饭都可以）
   - 不要过度P图，真实最重要
   - 不要全是怼脸自拍，多场景展示你的生活

5. **标签怎么加**
   - 必须加：#同城交友 #[城市]交友 #脱单 #[城市]脱单
   - 再加1-2个兴趣标签：#喜欢爬山 #美食探店 之类
   - 标签总共3-5个就够，不要加太多

6. **发布技巧**
   - 发布时间：工作日晚7-9点，周末下午2-5点，在线人数多
   - 定位：开定位，定在你所在城市，更容易被同城刷到
   - 互动：有人评论要及时回，互动率高推荐量也高
   - 如果没流量，可以删除隔天重发，换几张照片

7. **违规注意事项**
   - 不要直接放微信二维码/微信号，会被禁言封号
   - 不要发低俗内容，会违规
   - 不要导流到其他平台，第一次警告，第二次封号
   - 正确做法：先在小红书私信聊几句，感觉合适引导加微信"小红书不常看，加个微信聊吧"
